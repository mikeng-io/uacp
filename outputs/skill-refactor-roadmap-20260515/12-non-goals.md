# Non-Goals

This roadmap does not:

- fix all UACP docs immediately
- rewrite UACP doctrine wholesale
- modify protected UACP state
- claim lifecycle completion
- create one comprehensive SOP
- bulk-migrate all references in one step
- design final schemas for every phase upfront
- use UACP to validate itself
- delete old references without classification
- implement before preserving detailed decisions

The immediate goal is preservation and controlled sequencing, not heroic repair.
