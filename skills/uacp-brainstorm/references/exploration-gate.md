## Skill-Level Exploration Gate

This is a discipline rule inside `uacp-brainstorm`, not a UACP runtime component.

Do NOT do any of the following during brainstorming:

- Write implementation code
- Scaffold projects
- Modify production files or state
- Commit to version control
- Deploy anything
- Invoke implementation skills

**Allowed:**

- Ask questions
- Read files and docs
- Sketch approaches
- Write rough notes
- Produce the selected-scope package for TRIAGE

Brainstorming is exploration. Implementation starts only after TRIAGE admits the work and the lifecycle advances to PLAN/EXECUTE.
